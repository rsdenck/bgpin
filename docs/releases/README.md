# Release Notes

This directory contains detailed release notes for all bgpin versions.

## Available Releases

### [v0.2.0](v0.2.0.md) - March 3, 2026
**NetFlow/sFlow/IPFIX Collector + Vendor Parsers**

Major features:
- Complete NetFlow/sFlow/IPFIX collector implementation
- Real-time flow aggregation and BGP correlation
- Cisco parser (IOS/IOS-XE/IOS-XR/NX-OS)
- Juniper parser (JunOS with NETCONF)
- Anomaly detection
- GitHub Actions CI/CD for Linux builds

### [v0.1.0](v0.1.0.md) - March 3, 2026
**Initial Release**

Features:
- Complete RIPE RIS SDK
- CLI with ASN and Prefix commands
- Professional UX with go-pretty/table
- OpenTelemetry integration
- Multiple output formats (table, JSON, YAML)
