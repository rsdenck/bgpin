# Release Notes v0.3.0

**Release Date:** March 3, 2026  
**Tag:** v0.3.0

## New Features

### AI Integration
- **Multiple LLM Providers**: OpenAI, Claude, Gemini, Ollama
- **BGP Analysis**: AI-powered route analysis with security insights
- **Portuguese Support**: All AI responses in Portuguese
- **Interactive Copilot**: Chat-based BGP assistance
- **Rate Limiting**: 10 requests per minute per provider
- **Retry Logic**: Exponential backoff for network failures

### RPKI Validation
- **Route Validation**: Validate BGP routes against RPKI ROAs
- **RIPE Integration**: Uses RIPE RPKI Validator
- **Multiple Formats**: Table, JSON, YAML output
- **VRP Display**: Shows matched Valid Route Origin Authorizations

### MRT Parser
- **MRT File Support**: Parse Multi-threaded Routing Toolkit files
- **Historical Analysis**: Analyze historical BGP data
- **Route Dumps**: Process BGP table dumps
- **Update Messages**: Parse BGP update messages

### Enhanced Parsers
- **Arista EOS**: Complete parser implementation
- **Nokia SR OS**: Full support for Nokia routers
- **HTTP Looking Glass**: Generic HTTP-based looking glass parser

### Configuration Generator
- **Multi-Vendor**: Cisco, Juniper, MikroTik support
- **BGP Templates**: Generate BGP neighbor configurations
- **Automated Setup**: Streamlined router configuration

## AI Commands

### bgpin ai analyze
Analyze BGP prefixes with AI insights:
```bash
bgpin ai analyze 8.8.8.0/24 --provider ollama
```

### bgpin ai explain
Get human-readable explanations:
```bash
bgpin ai explain 1.1.1.0/24 --provider openai
```

### bgpin ai copilot
Interactive BGP assistant:
```bash
bgpin ai copilot
```

## RPKI Commands

### bgpin rpki validate
Validate routes against RPKI:
```bash
bgpin rpki validate 15169 8.8.8.0/24
```

## Configuration Commands

### bgpin config generate
Generate router configurations:
```bash
bgpin config generate bgp --vendor cisco --asn 65010 --neighbors 192.0.2.1 --remote-as 65020
```

## Configuration

### Environment Variables
- `OPENAI_API_KEY`: OpenAI API key
- `OPENAI_BASE_URL`: OpenAI API base URL (default: https://api.openai.com/v1)
- `ANTHROPIC_API_KEY`: Claude API key  
- `ANTHROPIC_BASE_URL`: Claude API base URL (default: https://api.anthropic.com/v1)
- `GOOGLE_API_KEY`: Gemini API key
- `GOOGLE_BASE_URL`: Gemini API base URL (default: https://generativelanguage.googleapis.com/v1beta)
- `OLLAMA_HOST`: Ollama server URL (default: http://10.1.254.32:11434)

### Provider Configuration
Each AI provider supports:
- Custom timeouts
- Model selection
- Rate limiting
- Retry configuration
